Complex Type: SiteMorphologyType

Documentation
Site Indicators related to the morpohology of a site, as suggested by SERA.

These include:
  • EC8 class
  • Seismological Bedrock Depth
  • Engineering Bedrock Depth (H800)
  • Geological Unit
For each site indicator network operators can provide at least the following information. For the allowed values in each case check the respective type.
  • Value
  • Reference
  • Quality Index

More information at: http://www.itsak.gr/SERA_Deliverable7.1.pdf.

Elements of <SiteMorphologyType>

Name Type Required Description
morphology MorphologyType optional Qualitative description of the shape of the earth's surface. Takes specific values.

Note:This in not included in the QuakeML 2.0 draft

Example: Slope

siteClassEC8Qindex1 RealQuantity optional
siteClassEC8Reference ReferenceType optional
bedrockDepth RealQuantity optional
bedrockDepthQindex1 RealQuantity optional
bedrockDepthReference ReferenceType optional

h800Reference ReferenceType optional
geologicalUnit Locally-defined simple type optional Free text description of the surface geology of a site

geologicalMapScale xs:string optional Scale of geological map used for the description of surface geology provided in geologicalUnit
